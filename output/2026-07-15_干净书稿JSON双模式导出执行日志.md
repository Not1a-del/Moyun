# 干净书稿 JSON 双模式导出执行日志

日期：2026-07-15
状态：已发布并完成 Pages 核验

## 用户要求

- 将网页版 JSON 导出改为两种干净模式：
  - 完整导出：世界观与正文；
  - 仅导出正文：正文且包含章节名。
- CoT、原生思考、工具记录、预设、流水线、文风、MOD、插件私有数据、图片及其他配置均不得导出。
- 本地测试后展示实际导出样本，再由用户确认上线。

## 变更边界

- 仅修改 Pages 仓 `source/moyun.single.html`、由其构建的静态产物与本日志。
- 不修改冻结离线区，不修改连接中心、生成逻辑、MOD 运行时或现有存档。

## 实施方案

1. 导出数据改为严格白名单，不再克隆整个 `book` 对象。
2. 两种模式都只保留导出元数据、书名、章节 `title/content`；完整模式额外保留 `novel.worldView`。
3. 正文走既有 `cleanNarrativeChapterContent`；世界观只移除明确 CoT / thinking 标签，不改写正常设定文字。
4. 用带有 CoT、预设、MOD 私有资料和模板内容的测试书走真实下载，检查字段、文本净化和导入兼容。
